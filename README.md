# red-bull-simulator-leaderboard
A high-performance Python application for real-time telemetry logging and competitive leaderboard management during the Red Bull Racing Simulator event.
## Overview
A standalone Python-based data acquisition tool designed to track, log, and rank participant performance in real-time during high-traffic simulator events. Developed for the **Road Arrow Formula Student Team** to enhance engagement at the International Belgrade Car Show.

## Key Features
- **Real-time Data Logging:** Captures participant data and lap times with zero latency.
- **Automated Ranking:** Dynamic leaderboard logic that sorts and displays top performers instantly.
- **Robust Persistence:** Implements a CSV-based flat-file database for lightweight and reliable data storage.
- **Standalone Deployment:** Compiled into a portable `.exe` for seamless cross-platform execution without Python environment dependencies.

## Tech Stack
- **Language:** Python 3.x
- **Libraries:** customtkinter, pandas, PIL
- **Deployment:** PyInstaller

## Impact
Successfully processed **100+** unique entries over **7** days of the exhibition, maintaining 100% data integrity and zero system downtime.
